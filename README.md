## 🖥️ System Features

*   **Retro Taskbar Navigation:** Seamless anchoring across desktop window components.
*   **Interactive Close controls:** Users can "close" widgets, triggering diagnostic logs in the simulated terminal.
*   **Simulated Diagnostic Console:** A terminal script displaying system configuration details in real time.
*   **Responsive Layout:** Neo-brutalist grid cards optimized for mobile, tablet, and widescreen monitors.

## 🚀 Running Locally

Launch the static environment using any web server. For example, using Python's built-in HTTP server:

```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000` in your web browser.
